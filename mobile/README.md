# Prefeitura Mobile

Aplicativo React Native (Expo) para iOS e Android.

## Rodando

```bash
cd mobile
npm install
npm run start        # abre o Expo Dev Tools
# ou diretamente:
npm run android      # dispositivo/emulador Android
npm run ios          # simulador iOS
```

## Estrutura
- `App.js`: tela de login com campos de e-mail e senha.
- `app.json`: configurações do app Expo (nome, ícones e splash).
- `assets/`: ícones e splash placeholders (substitua pelos definitivos).
